# Categories Resources

    POST api/categories/{id}/questions

## Description
Creates a question for a given category

It requires authoritazion (access token)

***

## Parameters

```
{
  "question": "Did Neymar score two goals against Metz?",
  "options": ["Yes","No"],
  "correctOption": "No",
  "date": "2017-08-18"
}
```
